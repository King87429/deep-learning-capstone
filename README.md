# Deep Learning Capstone

This repository contains implementations of core deep learning models developed as part of a capstone assignment.

## Included Experiments

### 1. Feedforward Neural Network
- Income Classification using Adult Dataset

### 2. Convolutional Neural Network (CNN)
- Fashion-MNIST Clothing Image Classification

### 3. Long Short-Term Memory (LSTM)
- Sequence Modeling / Time-Series Forecasting

## Technologies Used

- Python
- TensorFlow / Keras
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
